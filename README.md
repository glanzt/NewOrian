# מסע האוריינות - Literacy Quest 📚

מערכת למידה לבתי ספר שמלמדת ילדים בכיתה ב׳ אוריינות עברית דרך משחקים קצרים, ממכרים ומדידים.

## 🎯 מסלולים

המערכת מחולקת ל־4 מסלולים:

1. **📖 דיוק ושטף קריאה** - מעבר מקריאה בסיסית לקריאה שוטפת ומדויקת
2. **🔍 הבנת הנקרא** - פרטים → רצף → סיבה/תוצאה → מסקנה
3. **✏️ כתיבה מובנית** - הרכבת משפטים/פסקאות ללא הקלדה
4. **💎 אוצר מילים** - הקשר, נרדפות, הפכים, משפחות מילים

## 🚀 התחלה מהירה

```bash
# התקנת תלויות
npm install

# הפעלת שרת פיתוח
npm run dev

# בנייה לייצור
npm run build
```

## 🎮 סוגי משחקים

### מסלול קריאה
- **Sound Match** - האזנה ובחירה
- **Mistake Hunter** - איתור שגיאות
- **Reading Sprint** - קריאה מהירה

### מסלול הבנה
- **Story Detective** - שאלות הבנה
- **Sequence Builder** - סדר אירועים
- **Choose Ending** - בחירת סיום עם הוכחה

### מסלול כתיבה
- **Sentence Builder** - הרכבת משפטים
- **Paragraph Builder** - התחלה-אמצע-סוף
- **Fix Sentence** - תיקון שגיאות

### מסלול אוצר מילים
- **Gold Word** - מילה מהקשר
- **Opposites Arena** - הפכים ונרדפות
- **Word Family** - משפחות מילים

## 📊 מערכת התקדמות

- **XP** - נקודות ניסיון לכל תרגול
- **רמות** - 10 רמות לכל מסלול
- **רצף יומי** - תרגול יומי שומר על רצף 🔥
- **תגים** - פרסים על הישגים

## 🛠️ טכנולוגיות

- React 18 + TypeScript
- Vite
- TailwindCSS
- Framer Motion
- Zustand (state management)
- React Router

## 📝 תכנית שנתית

- נוב׳-דצמ׳: שיעורים ברמה בינונית
- ינו׳-מרץ: שיעורים מתקדמים
- אפר׳-יוני: טורנירים וסיכום
